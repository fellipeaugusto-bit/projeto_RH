# Sistema de Gestão de RH Completo

Este é um sistema de gestão de Recursos Humanos desenvolvido em Python, utilizando conceitos avançados de Programação Orientada a Objetos (POO) e integração com banco de dados MySQL.

## Conceitos de POO Aplicados

- **Encapsulamento**: Atributos protegidos e uso de @property (getters/setters).
- **Herança**: `Funcionario` herda de `Pessoa`; `Gerente` herda de `Funcionario`.
- **Polimorfismo**: Sobrescrita do método `mostrar_detalhes` nas subclasses.
- **Abstração**: Classe `Pessoa` definida como uma classe abstrata (ABC).
